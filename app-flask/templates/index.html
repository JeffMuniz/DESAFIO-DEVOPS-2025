# Desafio DevOps 2025

Este projeto contém duas aplicações em linguagens distintas (Flask e Node.js), cada uma com rotas para texto fixo e hora do servidor, integradas ao Redis com cache expirado e monitoradas via Prometheus e Grafana.

- Flask: http://localhost:5000
- Node.js: http://localhost:3000

Como rodar:
docker-compose up -d
